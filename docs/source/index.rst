PHAROST Documentation
=====================

PHAROST is a deep transfer learning framework for pharmacogenomic outcome prediction in spatial transcriptomics.

Overview
--------

PHAROST integrates pharmacogenomic knowledge with spatial transcriptomics to model drug sensitivity within tissue architecture. It enables high-resolution mapping of spatial drug-response heterogeneity.
